# AWS CLI

## Install
### Debian/Ubuntu
* We have 2 version
    * Old awscli-v1
    
[<img src="https://i.imgur.com/bJHu9QS.png">](https://i.imgur.com/bJHu9QS.png)

* And new awscli-v2:
````bash
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
````
