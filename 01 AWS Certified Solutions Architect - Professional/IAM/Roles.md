# Roles

## Acronym
* STS - Security Token service

## Desc
* Roles: short-term credentials, uses STS
    * Cross Account roles
    * Service Role: API Gateway, CodeDeploy, etc
    * EC2 Instance Roles: use the **EC2 metadata** service. One role at time per instance
