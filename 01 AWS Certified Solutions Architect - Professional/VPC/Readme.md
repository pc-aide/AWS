# VPC

## Acronym
* VPC - Virtual Private Cloud
* SG - Security Group
* NACL - Network Access Control List

## Core Components
* Subnets
* Route Tables
* Endpoints
* Gateways (if you want to get outside of your VPC)
  * NAT Gateways
  * Internet Gateways
  * Endpoint Gateways
* SGs
* NACLs (Firewall rules)
