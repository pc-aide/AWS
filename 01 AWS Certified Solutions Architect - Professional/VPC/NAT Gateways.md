# NAT Gateways

## Acronym
* NAT - Network Address Translation
* HA - High Available

## NAT
1) NAT gateway
  * HA per zone
  * Bandwidth up to 45Gbps
2) NAT instance
  * Use script to perform failOver
  * Bandwidth dependent on instance type
    * e.g -> t2.micro -> bandwidth -> **Low to Mederate**
